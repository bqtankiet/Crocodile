🛠️ Hướng dẫn triển khai Crocodile

Dự án Crocodile có thể được triển khai dễ dàng lên Tomcat server bằng cách build thành file .war.

1️⃣ Bước 1: Build file WAR

Sử dụng IDE IntelliJ IDEA để build file .war của ứng dụng.

mvn clean package

File WAR sẽ nằm trong thư mục target/ có tên là crocodile-1.0-SNAPSHOT.war

2️⃣ Bước 2: Đổi tên file WAR

Để Tomcat tự động triển khai ứng dụng ở http://localhost:8080 mà không cần chỉ định đường dẫn con, hãy đổi tên file:

crocodile-1.0-SNAPSHOT.war → ROOT.war

3️⃣ Bước 3: Copy vào thư mục webapps

Copy file ROOT.war vào thư mục webapps đã được mount tại:

docker/volumes/tomcat_webapps/ROOT.war

✅ Sau khi deploy

Sau khi deploy thành công, bạn sẽ thấy folder ROOT xuất hiện

Truy cập ứng dụng tại: http://localhost:8080

Ghi chú: Nếu bạn không đổi tên file thành ROOT.war, bạn sẽ cần truy cập theo tên file gốc, ví dụ: http://localhost:8080/crocodile-1.0-SNAPSHOT và phải sửa đổi lại một số đoạn code trong project.
